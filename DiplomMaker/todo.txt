Добавить стили:
	-	Обычный текст
	-	Заголовки:
		-	1, 2, 3, 4 уровня
		-	список или нет
		-	идёт после заголовка или нет
	-	изображение
	-	текст после изображения:
		-	перед текстом и перед заголовком (проверить точнее в документации)
	-	формула
	-	название таблицы
	-	заголовки таблицы
	-	обычные поля таблицы
	-	списки:
		-	а) б) в)
		-	1) 2) 3)
			- 1.1) 1.2) 1.3)
		-	1 2 3
			-	1.1 1.2 1.3
		-	-
		-	*

'@' - заголовок 1 уровня без списка: '@@' - второго без списка.
'#' - заголовок 1 уровня со списком: '##' - второго со списком.

Изображения можно вставлять по ссылке: '!img(URI) №номер'
Это загрузит изображение по ссылке и вставит в документ на это место. 
'№номер' будет заменено на 'Рисунок 1' или 'Рисунок А.1', но с учётом всех предыдущих номеров рисунков и приложений.
Также все ссылки на этот рисунок в тексте будут заменены на "см. рис. 1" и тому подобное.

Формула: '!formula(формула) №номер'.
То же самое, что с изображением, но с фомулой.
Текст после формулы идёт обычный, должен начинаться с "где".

func MakeDoc(text: String)
	lines = parseText(text)
	foreach line in lines:
		if line is №1 heading:
			add page breaker
			add line as heading
			
		else if line is heading:
				
			if previousLine is heading:
				add line as heading after heading
			else
				add line as heading after text
		
		else if line is dashlist:
		
			add line as dashlist
			
		else if line is after table:
			
			if nextLine is heading:
				add line as lineAfterTableBeforeHeading
			else
				add line as lineAfterTableBeforeText
end func
